Hello my dear Friends😁,

  this program is written with the aim of controlling temperature and humidity and turning on a device connected to a relay (here the relay is used alone) and you can use the DHT11 temperature and humidity sensor and ESP 32-s on the WROOM Adaptor and a Relay SRD using Flutter and HTTP protocol.
  
  The mechanism is that by executing the code of the DHT11_controller_esp.ino file in the link below and selecting the WROOM-DA board, you can easily run this project By closing the circuit as follows, your hardware components will be fully available😉❤️
